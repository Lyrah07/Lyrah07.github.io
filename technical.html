HTML

Hypertext Markup Language (HTML) is the standard markup language used to create the structure and content of web pages. It provides tags and attributes that define the various elements on a web page, such as headings, paragraphs, images, links, forms, and more.

HTML5 is the latest version of HTML, and it introduces new features and improvements over its predecessors.

HTML5 also includes new elements like <header>, <nav>, <article>, <section>, <footer>, which help in structuring the content more semantically.

CSS

Cascading Style Sheets (CSS) is used to style and layout the visual presentation of HTML documents.

It allows you to control the colors, fonts, spacing, and positioning of elements on a web page.

CSS is applied either internally within an HTML document, externally in a separate stylesheet file, or inline within HTML tags.

JavaScript

JavaScript is a scripting language that enables dynamic and interactive behavior on web pages.

  • It can manipulate the content and structure of HTML, handle user input, and communicate with servers (Ajax).
  • It is commonly used to create features like sliders, form validation, pop-up messages, and more.
  • It can be included directly within HTML documents or linked externally.
Git

Git is a free and open-source distributed version control system, specifically designed to manage projects of varying sizes efficiently and with speed. It offers several advantages over traditional SCM (Source Code Management) tools like Subversion, CVS, Perforce, and ClearCase. These include lightweight branching, convenient staging areas, and support for multiple workflows.

Github

GitHub is a website and cloud-based service that helps developers store, manage and collaborate on their code. GitHub uses Git software, which is a version control system that tracks and controls changes to the code.

VSCode

Visual Studio Code, commonly known as VSCode, is a popular source code editor developed by Microsoft. It's renowned for its versatility, extensibility, and efficiency, making it a favorite among developers worldwide.

Visual Studio Code is a free, lightweight and extensible code editor for building web, desktop and mobile applications. Available on Linux, macOS and Windows platforms

Linux

Linux is a family of open-source Unix-like operating systems based on the Linux kernel, an operating system kernel first released on September 17, 1991, by Linus Torvalds. It is typically packaged in a Linux distribution.

Linux is widely used in servers, mainframes, and supercomputers but is also prevalent in personal computers, mobile devices, embedded systems, and various other platforms. It's known for its stability, security, and flexibility, and it's popular among developers and system administrators due to its open-source nature, which allows for extensive customization and modification.

Why Linux

Linux is extremely versatile and is used for a wide range of applications, some of the most common:

  • Servers - Linux is a dominant force in the server world, powering web servers, database servers, email servers, and pretty much any other kind of server you can imagine. Its stability, security and customizability make it ideal for these critical tasks.
  • Supercomputers - The world's most powerful supercomputers almost always rely on Linux to handle massive amounts of data and complex calculations.
  • Web Development - Linux is a favorite among web developers for its powerful tools and flexibility. Many web hosting providers also use Linux.
